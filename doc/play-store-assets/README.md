# Play Store Assets

This folder contains the canonical assets used for the Google Play listing.

```
play-store-assets/
├── icons/
│   ├── app-icon-512x512.png
│   └── adaptive-icon-preview.png
├── feature-graphic/
│   └── feature-graphic-1024x500.png
└── screenshots/
    └── phone/
        ├── screenshot-01-launch.png
        ├── screenshot-02-initial-level.png
        ├── screenshot-03-interstitial.png
        └── screenshot-04-advanced-level.png
```

Generate screenshots with:
- `./build --screenshots`
