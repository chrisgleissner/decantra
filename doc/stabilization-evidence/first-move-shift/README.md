# First interaction no-shift evidence

This folder contains artifacts demonstrating that bottle rows 1 and 2 do not shift on first gameplay interaction.

## Visual evidence

- `before_first_move.png` — frame before the first interaction
- `after_first_move.png` — frame immediately after the first move interaction

## Positional evidence

- `drag_release_report.json` — PlayMode regression report generated by `FirstDragRelease_DoesNotShiftBottlePositions`
- Report values for row 1 (`Bottle_1`..`Bottle_3`) and row 2 (`Bottle_4`..`Bottle_6`) all show `deltaY: 0.0000`
- `maxDeltaY` is `0.000000`

## How to regenerate

1. Run PlayMode tests to regenerate drag-release positional report.
2. Run screenshot capture flow (`./build --screenshots` or `./build --screenshots-only`) to regenerate interaction images.
